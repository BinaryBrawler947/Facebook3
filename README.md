三、跨境选品黄金公式
▍爆款基因检测模型
市场热度 × 供给缺口 ÷ 运营难度 ≥ 3.5分

其中：

市场热度 = Google搜索量 ×0.6 + YouTube观看量 ×0.4
供给缺口 = 亚马逊评论数差异值 + 独立站新品率
运营难度 = (物流成本+广告CPC) × 利润率倒数

import requests
from bs4 import BeautifulSoup
def get_amazon_hot(category):
    url = f"https://www.amazon.com/bestsellers/{category}"
    headers = {'User-Agent':'Mozilla/5.0'}
    response = requests.get(url, headers=headers)
    soup = BeautifulSoup(response.text, 'html.parser')
    products = soup.select('#zg-ordered-list li')
    return [p.select_one('span.a-text-normal').text for p in products]

# 复古相机定价公式  
基础价 = eBay最近成交均价 × 0.8  
增值项：  
+ 20%（原装镜头保存完好）  
+ 15%（附带当年使用手册）  
+ 30%（有名人使用故事）  
最终定价 = (基础价 + 增值项) × 区域稀缺系数
